# Hospital_ER_Analysis_dashboard
1. Hospital Analytics : Analysis of Different Factors of Patients
An interactive and dynamic dashboard built to undestand the different factors of paients across different department in a hospital.

2. Short Description
This file has 2 dashboards. The first dashboard shows which month the patients visit the ER most, which age group of people come to the hospital and which department has the most patients etc.. And the second dashboard shows the analysis of wait time and satisfaction of patients. This tool is intended for use by medical analysts, and data-driven strategists who seek to comprehend the trend and make decisions based on that.

3. Tech Stack
The dashboard was built using the following technologies:
PowerBI - for visualization and report creation.
PowerQuery - for data transformation and cleaning.
DAX(Data Analysis Expressions) - for calculated measures and dynamic visuals.
File Format - .pbix for development and .png for dashboard previews.

4. Data source
Source: https://www.kaggle.com/datasets/laxdippatel/hospital-emergency-room-dataset
Data on - 9216 patients including their age, admission date, race, gender, satisfaction status and department referral.

5. Features
Business Problem: The Hospital management team wants to improve the institution's performance by analyzing different factors and making arrangements based on that so that patients are more satisfied with the hospital.
Goal of the Dashboard : The goal is to help hospital administrators identify bottlenecks in patient intake, average wait times and delays, high-risk time windows and areas requiring operational improvements.
Walkthrough of Key Visuals:
Main KPIs: Avg Wait Time, Avg satisfaction Score and No Response in Percentage.
Visuals: Wait Time Trend, Patients by Department, Satisfaction Score Breakdown, Gender and Age Demographics, and Department Performance comparison.
Insights: The highest ER visits came from Young Adult group. The longest average wait time came from the physiotherapy department.
